Minecraft AFK bot

Setup Guide

What you need to host this yourself.

Back to dashboard

1. Configure

Set your server and login

Use `.env` or `settings.json` to fill in the Minecraft host, port, username, and auth type.

2. Install

Install dependencies

Run `npm install`, then start with `npm start`.

3. Host it

Use a server, not your PC

If your computer is off, the bot must be running on a VPS, Render, Railway, Replit, or another always-on host.

4. Open dashboard

Control from the browser

Visit the app URL to start, stop, view logs, and send Minecraft commands.
